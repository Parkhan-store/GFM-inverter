# 20 kW Q-f Swing-Based Grid-Forming Inverter Control for Grid Interconnection
Simulation study of a 20 kW grid-forming inverter control for smooth grid interconnection.

## Overview
This project presents a simulation-based study of a 20 kW grid-forming inverter using Q-f swing-based control and P/Q reference scheduling for smooth grid interconnection.

The main objective is to achieve a stable transition from islanded operation to grid-connected operation under low-voltage microgrid conditions.

## Key Features
- 20 kW grid-forming inverter simulation
- Q-f swing-based virtual inertia control
- P/Q reference scheduling for grid interconnection
- Islanded-to-grid-connected transition
- Low-voltage microgrid operation
- PSIM-based power circuit and control simulation
- C-block implementation of synchronization and scheduling logic

## System Configuration
| Item | Value |
|---|---|
| Rated Power | 20 kW |
| DC-link Voltage | 700 V |
| Grid Frequency | 60 Hz |
| Switching Frequency | 20 kHz |
| Filter | LCL filter |
| Simulation Tool | PSIM, MATLAB/Simulink |

## Control Structure
The control system consists of:
- Grid-forming voltage control
- Q-f swing-based frequency generation
- P/Q reference scheduling
- Grid synchronization logic
- Smooth transition control for grid interconnection

## Simulation Results
Simulation results include:
- Inverter output voltage and current
- PCC voltage
- Active and reactive power response
- Frequency response
- Transition behavior during grid interconnection
